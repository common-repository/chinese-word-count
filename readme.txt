This plugin makes the WordPress 2.7 word count work with multibyte languages.

== Description ==

This plugin makes the WordPress 2.7 word count work with Chinese, Japanese and Korean.

Now compatible with WP 2.7.

这个插件使得WordPress 2.6在撰写博客文章时，可以正确统计中文字数。


== Installation ==

1. Upload `chinese-wordcount` directory to the `/wp-content/plugins/` directory
将`chinese-wordcount` 目录上传到 `/wp-content/plugins/` 目录
2. Activate the plugin through the 'Plugins' menu in WordPress
启用插件
